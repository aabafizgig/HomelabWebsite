Setting up Self-Hosted Web Server

I started today's ventures by installing a VM for this web server that this site is hosted on. I used Ubuntu Server as the base image and configured it with Nginx and SSH. This process was smooth and straightforward.

I downloaded Rufus and the Ubuntu ISO to create a bootable drive to use on the OptiPlex I acquired. I plan to install pfSense on it to use as a dedicated machine for my network-wide firewall later, but I wanted something other than Windows 7.

I received some of the cables I ordered today: Cat 6 Ethernet cables, a VGA cable that it turns out I did not need right now, but it is always good to have a spare on the shelf, or in the box, if you're like myself.
